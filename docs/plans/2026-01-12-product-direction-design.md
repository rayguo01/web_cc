## 一、产品定位

### 目标用户

**X 平台新蓝V用户**

- 为参与 X 创作者计划而订阅蓝V（$8/月）
- 需要满足 500 蓝V关注的条件，通过互关积累
- 已有付费意愿，但缺乏创作经验
- 面临选题、写作、时间、运营全方位挑战

### 核心价值

| 价值点 | 说明 |
|--------|------|
| 热点驱动 | 自动追踪 X 和国内热点，不用自己想选题 |
| 爆款方法论 | 内置 Defou x Stanley 方法论，提升内容质量 |
| 端到端闭环 | 热点抓取 → 内容生成 → 爆款优化 → AI 配图 → 发布 |

### 差异化定位

- X 平台专精（非通用 AI 写作工具）
- 热点驱动（主动推送选题，非用户想主题）
- 完整工作流（一站式完成，无需多工具切换）

### 定价策略：Freemium + 订阅

| 层级 | 价格 | 功能 |
|------|------|------|
| **免费版** | ¥0 | 每天 1 篇生成 + 爆款优化，无配图，月上限 30 篇 |
| **Pro 版** | ¥79-99/月 | 无限生成 + 爆款优化 + AI 配图（额度待定） |

| 策略 | 实现方式 |

## 三、功能规划

### 当前已完成功能（v2.1.2）

| 步骤 | 功能 | 状态 |
|------|------|------|
| 热帖抓取 | X-trends + TopHub | ✅ 已完成 |
| 生成内容 | content-writer（Defou x Stanley） | ✅ 已完成 |
| 爆款优化 | viral-verification | ✅ 已完成 |
| 生成配图 | Gemini 图片生成 | ✅ 已完成 |
| 发布 | 复制内容 + 保存历史 | ✅ 已完成（手动） |

### 下一阶段功能优先级

| 优先级 | 功能 | 方案 | 成本 | 说明 |
|--------|------|------|------|------|
| **P0** | 自动发布到 X | X Free API | $0 | 1,500 帖/月额度 |
| **P1** | 数据分析 | 暂缓 | $0 | 用户自行查看 X 后台 |
| **P2** | 用户系统（付费） | 50 人验证后 | 待定 | 含订阅、额度、支付 |

### 自动发布技术方案

- **X Developer 账号**：已有
- **API 级别**：Free（1,500 帖/月，不能读取数据）
- **实现方式**：OAuth 授权 + X API v2 发帖接口

### 数据分析方案

暂缓实现，原因：
1. X Free API 无法读取帖子互动数据
2. 升级 Basic 需 $100/月，成本较高
3. 先验证核心价值（内容生成），再考虑增值功能

替代方案：用户自行在 X 后台查看帖子表现
