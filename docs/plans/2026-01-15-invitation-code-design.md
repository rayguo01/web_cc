# 邀请码功能设计

## 概述

为管理后台添加侧边栏导航，并实现邀请码注册系统。

## 管理后台侧边栏

三个导航项：
1. **统计概览** - 全局数据 + 用户消耗排行
2. **邀请码管理** - 生成、查看、筛选邀请码
3. **用户管理** - 用户列表 + "以该用户身份登录"功能

## 邀请码系统

### 邀请码格式
- 格式：`VX-XXXXXX`（VX- 前缀 + 6位大写字母数字）
- 示例：`VX-A8K2M9`

### 生成邀请码
管理员填写：
- 数量：1-100 个
- 有效期：7天 / 30天 / 90天 / 永久
- 备注：可选，记录用途

### 邀请码状态
- **可用** - 未使用且未过期
- **已使用** - 已被注册使用
- **已过期** - 未使用但超过有效期

### 邀请码列表
- 显示：邀请码、状态、备注、创建时间、使用者
- 筛选：全部 / 可用 / 已使用 / 已过期
- 支持搜索备注

## 注册流程变更

用户注册时新增"邀请码"必填项，验证通过才能完成注册。

## 用户管理特殊功能

管理员可以"以任意用户身份登录"系统，用于排查问题或代操作。
