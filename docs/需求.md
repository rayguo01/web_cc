# 项目需求
1.在服务器上启动一个web网页，用户可以输入内容
2.该内容传递给同一台服务器上启动的claude code终端，cc进行处理，再把返回的内容在网页端显示出来

# 项目方案
使用 Node.js 创建一个 Web 服务器，通过子进程调用 `claude` CLI，并通过 WebSocket 实现实时流式输出。

## 核心实现思路

```
浏览器 <--WebSocket--> Node.js服务器 <--子进程--> claude CLI
```

**关键命令**：Claude Code 支持非交互模式：
```bash
claude -p "你的提示词" --output-format stream-json
```


**完整版本** - 支持流式输出、会话管理、更好的UI

## 其他需求

- Node.js 实现
- 需要**保持会话上下文**
- 服务器环境是ubuntu
- 多用户系统，每个用户需要登陆
- 对话数据保存在数据库，数据库为neon
